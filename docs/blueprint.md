# **App Name**: Kivo

## Core Features:

- User Authentication: Secure user registration and login.
- Social Login: Allow users to sign in with Outlook or Google.
- Input Validation: Validate username and password inputs.
- Password Reset: Enable users to reset their passwords.
- Account Creation: Allow users to create a new account.

## Style Guidelines:

- Primary color: Light blue (#698FCA), used in the logo and main elements.
- Secondary color: White (#FFFFFF) for backgrounds.
- Accent color: Light grey (#F2F2F2) for input fields and subtle UI elements.
- Body and headline font: Use a clean, sans-serif font such as 'Roboto' or 'Open Sans' for readability.
- Font weight: Use a medium font weight for headlines and a regular font weight for body text.
- Use simple, outlined icons from a library like Material Design Icons for input fields and buttons.
- Center the login form on the page for a clean and focused presentation.
- Use a responsive layout that adapts to different screen sizes.
- Subtle transitions and animations for button hovers and form element focus.
